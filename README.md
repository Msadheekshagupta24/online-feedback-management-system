## 📌 Project Overview

This is a responsive web-based feedback and review management system that allows users to submit feedback and ratings. The system analyzes user sentiment (Positive, Neutral, Negative) to help understand user behavior and improve product experience.
## 🚀 Features

- ⭐ Star-based rating system  
- 📊 Sentiment classification (Positive / Neutral / Negative)  
- 📁 Filter and analyze feedback data  
- 📤 Export data as CSV  
- 💾 LocalStorage for persistent data
